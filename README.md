# ChatGuessr

> Play GeoGuessr on Stream with Twitch Chat. Interactive application

# Deploy

1. Run windows powerShell from Administrator
2. Execute [Environment]::SetEnvironmentVariable("GH_TOKEN","<git_hub_token>","User")
3. \$ npm run deploy
